󰜢 System Prerequisites

Prepare your local physical workstation infrastructure with the core toolsets needed to support the workspace framework:

  • Python Runtime Environment: Python 3.8+ configured with an integrated system pip module config tracker.
  • Local Model Processing Layer: A standalone Ollama server application running locally and exposed at http://localhost:11434.
  • LLM Weight Map: Download and host the lightweight, hyper-responsive gemma3:1b footprint signature locally.
  • Universal Browser Compatibility: Perfectly optimized for Chrome, Firefox, Edge, Safari, and all WebKit/Chromium privacy engines.
󰏔 Provisioning & Server Deployment

1. Clone the Central Repository

git clone https://github.com/excellogalihs/litelaw.git
cd litelaw

2. Install Core Packages & Drivers

# Installs the modern Flask server components, text document converters, and ddgs module tools
pip install flask pypdf python-docx pillow ddgs

3. Stage LLM Models Locally

# Fetch the target AI language parameters inside your local Ollama daemon
ollama pull gemma3:1b

4. Boot the Modern Workspace Engine

Execute the central deployment file to kickstart the graphical dashboard environment:

python3 app.py
