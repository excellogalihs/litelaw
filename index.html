Core Intelligence & Planning

Autonomous Agent Loop

THOUGHT → ACTION → COMMAND reasoning loop that executes terminal automation step by step and stops itself once the task is done.

Guardrails

Lightweight self-correction rules that catch common small-model mistakes before they reach your filesystem.

  • Blocks invalid command formatting
  • Automated command safety checks
  • Infinite loop & repetition detection

Persistent Memory

Long-term knowledge storage layer extending beyond reboots to enforce rules across localized tasks.

  • Maintains chosen coding conventions
  • Saves global project and task contexts
  • Injected into every new chat's system prompt

System Installation Guide

Deploy your private, SaaS-free local workspace environment onto your physical hardware within minutes.

1. Environment Prerequisites

requirements.txt
• Python 3.10 or higher
• Ollama Engine (Installed locally)
• Supported Local Architectures: gemma3:1b, gemma3:4b, llama3.2:1b, llama3.2:3b

2. Shell Implementation Steps

terminal — bash shell
# Clone the core architecture
git clone https://github.com/excellogalihs/litelaw.git
cd litelaw

# Provision local operational dependencies
pip install flask pillow python-docx pypdf duckduckgo_search

# Initialize and wake Ollama local server runtime
ollama serve

# Pull target model architecture matrix
ollama pull gemma3:1b

# Fire up your workspace environment
python app.py

Target Local Host: http://localhost:5000
