System Installation Guide

Deploy your private, SaaS-free local workspace environment onto your physical hardware within minutes.

1. Environment Prerequisites

requirements.txt
• Python 3.10 or higher
• Ollama Engine (Installed locally)
• Git (Optional for repository fetching)
• Recommended Foundation Architecture: gemma3:1b

2. Shell Implementation Steps

terminal — bash shell
# Clone the core architecture
git clone https://github.com/excellogalihs/litelaw.git
cd litelaw

# Provision local operational dependencies
pip install flask pillow python-docx pypdf ddgs

# Initialize and wake Ollama local server runtime
ollama serve

# Pull target model architecture matrix
ollama pull gemma3:1b

# Fire up your workspace environment
python app.py

Target Local Host: http://127.0.0.1:5000
